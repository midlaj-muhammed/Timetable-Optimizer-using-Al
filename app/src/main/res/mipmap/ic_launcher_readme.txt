App Icon Files:

The following icon files are required for the app launcher:

Required icon files:
- ic_launcher.png (48x48, 72x72, 96x96, 144x144, 192x192 dp)
- ic_launcher_round.png (same sizes)
- ic_launcher_foreground.xml (vector drawable)
- ic_launcher_background.xml (vector drawable)

Icon sizes for different densities:
- mdpi: 48x48 px
- hdpi: 72x72 px
- xhdpi: 96x96 px
- xxhdpi: 144x144 px
- xxxhdpi: 192x192 px

You can generate these icons using:
1. Android Studio's Image Asset Studio (recommended)
2. Online icon generators
3. Design tools like Figma, Sketch, or Adobe Illustrator

To generate icons in Android Studio:
1. Right-click on app/src/main/res
2. Select New > Image Asset
3. Choose "Launcher Icons (Adaptive and Legacy)"
4. Upload your icon design
5. Configure foreground and background
6. Generate all required sizes

Icon design guidelines:
- Use a simple, recognizable design
- Ensure good contrast
- Test on different backgrounds
- Follow Material Design icon principles
- Consider the app's purpose (timetable/scheduling)

Suggested icon concepts:
- Calendar with AI/brain symbol
- Clock with optimization arrows
- Grid/table with smart indicators
- Schedule/timeline with AI elements
